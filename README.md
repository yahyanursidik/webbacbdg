# BAC Company Profile

Website company profile, katalog produk, dan jalur order WhatsApp untuk CV. Berkah Abadi Creative.

## Stack

- Astro 7
- Static output untuk Vercel
- Sitemap XML via `@astrojs/sitemap`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Konfigurasi Konten

- Data brand dan kontak: `src/data/site.ts`
- Data kategori dan produk: `src/data/products.ts`
- Gambar lokal: `public/images`

Ganti `whatsappNumber`, alamat, Google Maps, dan foto produk dengan data asli BAC sebelum publish.
